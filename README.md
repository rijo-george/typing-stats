# Typing Stats

**Know how much you type. Every. Single. Day.**

Track keystrokes, words typed, and WPM throughout the day. Fun end-of-day summaries with comparisons, a 24-hour activity chart, an animated desktop buddy, and shareable summary cards. Native macOS menu bar + floating character app.

## Download

**[Download for Mac](https://github.com/rijo-george/typing-stats/releases/latest)** — Universal binary, signed & notarized. Works on Apple Silicon and Intel.

## Features

- **Real-time WPM** — Live words-per-minute in the menu bar.
- **Word & keystroke counting** — Every word, every keystroke, tracked all day.
- **Peak WPM tracking** — Your fastest burst captured and highlighted.
- **24h activity chart** — Hour-by-hour visualization of your typing activity.
- **Fun comparisons** — "You typed 12,847 words — that's 26 pages of a novel."
- **Shareable summary card** — Beautiful end-of-day card to share on social media.
- **Desktop Buddy** — Animated character that types along with you and evolves with your speed.
- **100% private** — No accounts, no tracking. All data stays on your Mac.

## Desktop Buddy

An animated character sits on your desktop and types along with you. The faster you type, the cooler it gets:

| WPM | Character |
|-----|-----------|
| 0–19 | Sloth |
| 20–39 | Turtle |
| 40–59 | Cat |
| 60–89 | Rabbit |
| 90+ | Cheetah |

## Building from Source

```bash
bash build-app.sh        # Build + sign
bash build-dmg.sh        # Build + sign + notarize DMG
```

Requires Xcode 16+ and macOS 14+.
